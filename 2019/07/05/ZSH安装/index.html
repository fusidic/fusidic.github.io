[NOTE] ZSH安装

习惯了Hyper+WSL+zsh的组合，在centos中也得安排上，备查。

Step 1: Install & Configure ZSH

CentOS 7：

yum install zsh

Ubuntu:

apt install zsh

Step 2: ZSH Settings

CentOS 7:

chsh -s /bin/zsh root

Ubuntu:

which zsh

Log in again and check the current shell by:

echo $SHELL

Step 3: Install & Configutre Oh my zsh

CentOS 7:

yum install wget git

Ubuntu:

apt install wget git

Download the installer script and run it by:

wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh

And this will meets a problem if the system is the simplified. It will be refused by githubusercontent.com for the lack of ca-certificates. And there are two ways to deal with it:

wget --no-check-certificate https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | zsh

or do it once:

yum install ca-certificates -y or yum install ssl-cert

After we have installed Oh my zsh in ~/.on-my-zsh, we just need to copy it in .zshrc and apply the configuration.

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

source ~/.zshrc

Try this:

sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
